# Soil Temperature Analysis Scripts

This repository contains all the main Python scripts and analysis used in my Master's thesis for evaluation of multiple Soil temperature (ST) products at surface and root-zone layers using in-situ networks from the International Soil Moisture Network (ISMN). **Data is not included**, only the code.

## Surface foler, rootzone_layer
These folders contain subfolders for each soil temperature (ST) product evaluated at the surface (0–10 cm) and root-zone (10–100 cm) layers. Each subfolder includes Python scripts for data downloading, exploratory analysis, temporal harmonization, spatial collocation, and pixel-mean computation, along with the output files generated by these scripts (analysis results).

## Excel_tables
This folder contains all error metric tables (bias, ubRMSD, correlation) for each ST product against each ISMN network, computed at both native resolution and temporally harmonized resolution (4 timestamps per day) for surface and root-zone layers.
